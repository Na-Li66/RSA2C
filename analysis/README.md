# Analysis Utilities

This directory contains environment-agnostic plotting and aggregation utilities
for experiment outputs generated by the training scripts.

| File | Purpose |
| --- | --- |
| `plot_returns.py` | Plot one or more return curves from CSV, NPZ, or directories of CSV files. |
| `plot_shap.py` | Plot SHAP heatmaps or beeswarm plots for Pendulum, Walker, and Ant via `--env`. |

Generated files such as CSV, NPZ, PNG, and PDF outputs are intentionally ignored
by `.gitignore`.
